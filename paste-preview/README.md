# Paste Preview

Instant paste and drag-and-drop preview for text, images, and files.

## Run

```bash
cd paste-preview
npm install
npm run dev
```

Open the URL shown in the terminal. Paste with **Ctrl+V** or drag files anywhere on the page.

## Supports

- Images — inline preview
- Text — monospace block
- Video / audio — native player
- PDF — embedded viewer
- Other files — icon card with download link

Press **Escape** to clear all. Click **×** on a card to remove one item.
